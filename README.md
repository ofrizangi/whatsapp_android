# Android Client:

A message app.
Developed by Sivan Jhirad & Ofri Zangi.

## How To Run?
1. pull all files from git to android studio 
2. Run the server WebWhatsappApi from the project - https://github.com/ofrizangi/Whatsapp_Server2
3. run the emulator
